## The AI Toolkit for Antigravity

Ag-Cel transforms your local environment into a powerhouse for AI agents. It bridges the gap between local tools and AI capabilities with structured Skills and Workflows.

<CardGrid>
  <Card title="Workflows" icon="rocket">
    Standardized procedures for common tasks like `/feature`, `/fix`, and `/deploy`.
  </Card>
  <Card title="Skills" icon="puzzle">
    Specialized capabilities for Languages, Frameworks, Security, and detailed Methodologies.
  </Card>
  <Card title="Rules" icon="setting">
    Guardrails and operational rules to ensure AI safety and consistency.
  </Card>
  <Card title="CLI" icon="terminal">
    Simple command-line interface to manage your local MCP server.
  </Card>
</CardGrid>

---

## Quick Start

```bash
# Install globally
npm install -g ag-cel

# Initialize in your project
agc init

# Start the server
agc start
```
